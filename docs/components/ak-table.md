# AkTable

Table component with head and body. Yields `t.head` and `t.body` with columns and rows.

## API Reference

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `headerColor` | string | - | Header background (e.g. neutral) |
| `variant` | string | - | full-bordered, etc. |
| `borderColor` | string | - | light or dark |
| `hoverable` | boolean | `false` | Row hover effect |
| `dense` | boolean | `false` | Compact rows |
